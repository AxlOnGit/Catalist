void PdfFileWriter.PdfChart.SaveToPdfFile ( Boolean  DisposeChart = true)

Save chart to PDF output file

Parameters
DisposeChartDispose .NET chart object (see remarks.)

Call this method after chart is fully constructed. Calling this method flush the resource to the output file. It reduces the unmanaged memory requirements. If method is not called, the resource will be sent to the output file when Document.CreateFile() method is called.

If DisposeChart is true (by default it is true), this method will call .NET Chart.Dispose() method to free unmanaged resources. If DisposeChart is false, it is the responsibility of the caller to free the resources.
