Call this method after chart is fully constructed. Calling this method flush the resource to the output file. It reduces the unmanaged memory requirements. If method is not called, the resource will be sent to the output file when Document.\+Create\+File() method is called. 

If Dispose\+Chart is true (by default it is true), this method will call .N\+ET Chart.\+Dispose() method to free unmanaged resources. If Dispose\+Chart is false, it is the responsibility of the caller to free the resources.
