\subsection{Detailed Description}
P\+DF print document class 

It is a derived class of Print\+Document. The class converts the metafile output of Print\+Document to an image. The image is displayed in the P\+DF document. 

For more information go to \href{http://www.codeproject.com/Articles/570682/PDF-File-Writer-Csharp-Class-Library-Version#PrintDocumentSupport}{\tt 2.\+11 Print Document Support}
